# Java Practice Programs

A collection of **Java practice programs** created to strengthen understanding of **core Java concepts**, problem solving, and programming fundamentals.

This repository contains programs covering topics such as operators, conditions, loops, functions, object-oriented programming, and basic Java logic building.

---

## Technologies Used

- Java
- OOP Concepts
- Scanner Class
- Conditional Statements
- Loops
- Methods & Functions

---

## Topics Covered

| Topic | Description |

## How To Run

### Using Command Prompt

Compile the Java file:

```bash
javac FileName.java
```

Run the program:

```bash
java FileName
```

### Example

```bash
javac Comparison.java
java Comparison
```

---

### Using IntelliJ IDEA / VS Code

1. Open the project.
2. Select the Java file.
3. Run the program.
4. View output in the console.
